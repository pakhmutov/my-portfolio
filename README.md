Personal portfolio site built with Next.js and TypeScript.

**[vpakhmutov.vercel.app →](https://vpakhmutov.vercel.app)**

## Stack

| | |
|---|---|
| Framework | Next.js 16 (App Router) |
| Language | TypeScript |
| Styles | SCSS (CSS Modules) |
| Deploy | Vercel |

## Features

- Custom cursor — dot + ring with lerp lag
- Intro loader — VP initials animation
- Particle background — 120 particles with mouse interaction
- Scroll-triggered fade-in animations
- Vercel Analytics

## Local setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
